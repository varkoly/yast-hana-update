Welcome to the SUSE SAP HANA Cluster Update

This wizard allows for upgrading SAP ® HANA ® System Replication (Scale-Up) instances in a SUSE cluster setup by performing a so-called Near Zero Downtime Upgrade.

The wizard handles the part of the procedure related to the SUSE cluster, while some of the steps have to be performed manually by the SAP HANA admin.

Before you begin:

  • Make sure that you have created a full backup of all HANA instances and that all cluster nodes have password-less SSH access to one another.
  • Make sure that you have created the user in the local HANA userstore under the key SRTAKEOVER and granted the required priviliges. Please refer to the section Use SAP HANA System Replication for Near Zero Downtime Upgrades of the SAP HANA Administration Guide for more information.
