Cluster Configuration

This step shows the generated update plan for the clustered SAP HANA system.

Please note that the Wizard handles only the cluster itself, and all SAP HANA-related steps are marked as [Manual step] in the plan.

Please also note, that in order to disable System Replication, the secondary SAP HANA instance has to be stopped first, and then started again.

Once you click Next, the Wizard will apply proposed changes to the cluster.

Click Skip to skip current step. Note: use this function with caution! Misusing this function can lead to an inconsistent cluster configuration.
